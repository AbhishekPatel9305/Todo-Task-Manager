# Todo Task Manager Web App

A simple and efficient task management web application that helps users add, delete, and manage daily tasks easily.

## Features
- Add new tasks
- Delete existing tasks
- Clean and responsive user interface
- Built with pure HTML, CSS, and JavaScript

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Run the Project
1. Download or clone the repository
2. Open the `index.html` file in any web browser

## Project Purpose
This project was created for practice and learning basic front-end development skills.
